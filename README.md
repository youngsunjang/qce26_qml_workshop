# qce26_qml_workshop
Code and notebooks for the QCE26 workshop paper:

**Parity Learning with Small Quantum Neural Models**

This repository contains experiments for an unsupervised 16-bit even-parity generative benchmark. The models include:

- Local-window AR-MLP
- Masked causal AR-CNN
- QCBM-MMD

Notebooks and results will be updated before the paper submission deadline.
